# Cold-chain-validation
冷链验证
系统由冷链验证平台、中继器、无线智能温度采集仪三部分组成，通过部署在冷链终端上的智能设备，实时采集终端各个环节的监测信息，无线连接到中继器，然后通过中继器将终端的温度信息发送到云端平台，用户可通过中继器和云端平台实现实时冷链终端监测和智能化管理。
